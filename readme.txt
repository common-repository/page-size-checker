Page Size Checker will help you to findout exact size of web page (including text and images) that loads in user computer. It is very helpful to manage bandwidth, page loading time and content management. 

== Description ==

Your website will load very slow if its content is big. Page Size Checker will help you to find exact size of web pages while it loads in user end. It is very easy, fast and powerful tool to check size for all pages of website. Keep in mind that more page size means more bandwidth. It also help you to manage your limited bandwidth as well as user data package. It also help your website to get better place in Search Engine. It is good tool for SEO.

<h4>How to use</h4>
<ul>
	<li>Install and active plugin </li>
	<li> When you want to use it, click on Page Size Checker from left menu </li>
	<li>You can see all pages and posts  listed there</li>
	<li>Click on individual Title which you want to check </li>
</ul>

<h4>How it works</h4>

This plugin just send your domain (URL) to http://seorankanalyser.com/ web site and get details data from server. All data processed in http://seorankanalyser.com and return results to your plugin's panel. 

== Installation ==

1. Download, unzip and upload to your WordPress plugins directory
2. Click on Page Size Checker from menu and get report from there
